# telega_stalker

## Обновленные правила по рейдам и войне

### Рейды по сложности

**Лайт:**
- рейд на слепых псов
- рейд на кабанов

**Средний:**
- рейд на кровососов
- рейд на снорков
- рейд на бандитов

**Хард:**
- рейд на химеру
- рейд на контролера
- рейд на бюрера

### Дополнительно по рейдам
- Подписывать/обозначать рейды как рейды на логово.
- Награды за рейды выдавать артефактами.
- Выдавать не более **2 артефактов за раз**.
- Для рейдов за артефакты установить сложность: **25 силы**.
- Рейд запускать от **2 человек**.

### Война
- Войну запускать от **5 человек**.

## Поиск артефактов (новая механика)

### Детекторы в снаряжении

Добавить в раздел снаряжения детекторы артефактов:

- **Отклик** — 1000 RU
- **Медведь** — 4000 RU
- **Велес** — 10000 RU
- **Сворог** — 30000 RU

### Базовые шансы найти артефакт

- **Отклик**: 10%
- **Медведь**: 20%
- **Велес**: 35%
- **Сворог**: 50%

### Формула поиска (предложение)

Итоговый шанс поиска артефакта считать так:

`Шанс = База_детектора + Бонус_снаряжения + Бонус_события - Штраф_локации`

Где:

- `База_детектора` — фиксированное значение от выбранного детектора (10/20/35/50).
- `Бонус_снаряжения = min(10, floor(Сила_снаряжения / 2))`.
- `Бонус_события` — от 0 до +10 (например, если на локации активна аномалия, повышающая шанс).
- `Штраф_локации` — от 0 до 15 (чем опаснее зона, тем выше штраф).

Ограничения:

- Минимальный итоговый шанс: **5%**
- Максимальный итоговый шанс: **85%**

Итог:

`Итоговый_шанс = clamp(5, 85, Шанс)`
